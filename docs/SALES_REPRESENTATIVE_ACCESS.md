# Sales Representative Dashboard Access Guide

## Overview
The Sales Representative Dashboard provides a comprehensive view for sales staff to manage their orders, customers, returns/exchanges, complaints, and performance metrics.

## How to Access

- You must have a user account with the "Sales Representative" role
- Log into the system at `/login`

## Role-Based Access
- Only users with "Sales Representative" role can access this dashboard
- Each representative only sees their assigned orders, customers, and related data
- Performance metrics are personalized per representative

## Features
- Real-time data updates
- Responsive design for mobile/tablet access
- Export capabilities for reports
- Notification badges for pending items
- Interactive charts and analytics

## Navigation Tips
- Use the tab navigation to switch between sections
- Red badges indicate items requiring attention
- Use filters and search to quickly find specific data
- Click on items for detailed views and actions

## Support
If you encounter any issues accessing the dashboard, contact your system administrator to verify your role permissions.
